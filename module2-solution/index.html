Some Programming Languages

Python

A language most used for machine learning purposes and data science. It is very easy to use and is suggested for those who are just starting programming.

C ++

Prefered by a vast majority, it is a strong language for game developing and desktop aplications. Is more complex than others languages and some OS are written using it.

Java

With a large amount of libraries, is a powerful tool wich allows the user write from android apps to web aplications. It's versatility grant us the freedom to code whatever we want, but sometimes is not that easy to fix errors.
